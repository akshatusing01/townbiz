# TownBiz

## Project Overview
TownBiz is a comprehensive platform designed to connect local businesses with their communities, enabling seamless interaction between consumers and service providers. It aims to empower small businesses by providing them with the tools they need to thrive in a digital economy.

## Tech Stack
- **Frontend:** React.js, Redux
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Styling:** Tailwind CSS

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/akshatusing01/townbiz.git
   cd townbiz
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Create a `.env` file and add your configurations as per the example provided in `.env.example`.
4. Run the application:
   - For development mode:
   ```bash
   npm run dev
   ```
   - For production mode:
   ```bash
   npm start
   ```

## Features
- User Authentication: Secure login and registration.
- Business Listings: Search and filter local businesses.
- Reviews and Ratings: Customers can leave feedback for businesses.
- Event Calendar: Businesses can post events.
- Notifications: Users can receive updates about their favorite businesses.
